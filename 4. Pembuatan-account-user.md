Jika ingin menambahkan PPSK atau custom attribute:
- Pilih â€œInsert a custom attributeâ€
- Isi: `Custom Attribute : Tunnel-Password`
- Klik Add Attribute
- Isi kolom:
    - Attribute -> Tunnel-Password
    - Value -> (misal) 12345678
    - Op -> :=
    - Target -> reply

Lalu klik Apply.
